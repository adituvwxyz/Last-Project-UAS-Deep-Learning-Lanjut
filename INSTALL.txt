5Ô∏è‚É£ SETUP GROQ API KEY
=================================================================

üîë LANGKAH 1: Dapatkan API Key GRATIS
--------------------------------------
1. Buka browser: https://console.groq.com/
2. Klik "Sign Up" (daftar akun baru)
   ATAU "Sign In" (jika sudah punya akun)
3. Login dengan Google Account (GRATIS!)
4. Setelah login, klik menu "API Keys"
5. Klik tombol "Create API Key"
6. Copy API Key yang muncul
   Format: gsk_xxxxxxxxxxxxxxxxxxxxxxxx


üìù LANGKAH 2: Masukkan API Key
-------------------------------
1. Buka file: .env (di folder project)
2. Gunakan text editor (Notepad, VS Code, dll)
3. Cari baris:
   
   GROQ_API_KEY=gsk_your_api_key_here

4. Ganti "gsk_your_api_key_here" dengan API Key Anda
   Contoh:
   
   GROQ_API_KEY=gsk_abc123xyz789def456ghi

5. SAVE file (Ctrl + S)

‚ö†Ô∏è PENTING: Jangan share API Key ke orang lain!

8Ô∏è‚É£ TROUBLESHOOTING
=================================================================

‚ùå MASALAH 1: Chatbot Tidak Menjawab
-------------------------------------
SOLUSI:
1. Cek API Key di file .env
2. Pastikan format: GROQ_API_KEY=gsk_xxxxx
3. Restart server:
   - Tekan Ctrl+C di terminal
   - Ketik: npm start
4. Refresh browser (F5)


‚ùå MASALAH 2: npm install Error
-------------------------------
SOLUSI:
1. Hapus folder node_modules (jika ada)
2. Ketik: npm cache clean --force
3. Ketik lagi: npm install


‚ùå MASALAH 3: Port 3000 Already in Use
--------------------------------------
SOLUSI:
1. Edit file .env
2. Ubah PORT=3000 jadi PORT=3001
3. Jalankan lagi: npm start
4. Buka: http://localhost:3001


‚ùå MASALAH 4: Halaman Tidak Muncul
----------------------------------
SOLUSI:
1. Cek server running di terminal
2. Buka: http://localhost:3000 (bukan https)
3. Clear cache browser (Ctrl+Shift+Del)
4. Refresh (Ctrl+F5)


‚ùå MASALAH 5: Groq API Error
-----------------------------
SOLUSI:
1. Cek kredit API di https://console.groq.com/
2. Pastikan API Key benar (copy paste ulang)
3. Cek koneksi internet
